NoteFlow Mobile · Android

Privacy Policy

Last updated: April 12, 2026

Summary: NoteFlow Mobile stores all your notes locally on your device. We do not collect, transmit, or sell any personal data. No account required.

1. Who We Are

NoteFlow Mobile is an independent note-taking application for Android. For questions about this policy, contact us at yago.igle@gmail.com.

2. Data We Collect

We collect no personal data whatsoever.

NoteFlow Mobile does not:

  • Require an account or registration
  • Ask for your name, email, or any personal information
  • Track or log your usage, behaviour, or app interactions
  • Send analytics or telemetry to any server
  • Display advertisements
  • Share or sell any data to third parties

3. Data Stored on Your Device

All notes, groups, and preferences are stored exclusively on your device using the following local technologies:

  • SQLite — local database for notes and note groups
  • Expo SecureStore — encrypted storage for optional note-level encryption passwords

None of this data is transmitted anywhere unless you explicitly choose to share or export it using the built-in share function.

4. Optional GitHub Sync

NoteFlow offers an optional feature to sync your notes with a private GitHub repository that you own and control. This feature is entirely opt-in. If you enable it:

  • Your GitHub personal access token is stored locally using encrypted storage and never leaves your device except in API calls made directly to GitHub
  • Notes are synced directly between your device and your own GitHub repository — NoteFlow has no intermediary server
  • You can revoke access at any time by removing the token in the app settings or deleting it from your GitHub account

When using GitHub sync, GitHub's privacy policy applies to data stored in your repository.

5. Android Permissions

Permission Why it is needed
INTERNET Required only for the optional GitHub sync feature. Not used if sync is disabled.

(Android 12 and below)		 Allows importing note files from local storage.

6. Third-Party Services

NoteFlow Mobile does not integrate any third-party analytics, crash reporting, advertising, or tracking SDKs. The only optional external communication is the direct device-to-GitHub API call described in section 4.

7. Children's Privacy

NoteFlow Mobile is not directed at children under 13 years of age, and we do not knowingly collect any information from children.

8. Changes to This Policy

If this policy changes, the date at the top of this page will be updated. Continued use of the app after changes constitutes acceptance of the updated policy.
